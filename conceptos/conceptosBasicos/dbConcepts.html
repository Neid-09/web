Conceptos Básicos de Base de Datos

¿Qué es una base de datos?

Una base de datos (BD) es una colección estructurada de datos que deben reflejar fielmente los objetos, relaciones y restricciones del mundo real al que representan. Para que esta representación sea fiable, la base de datos debe ser capaz de evolucionar y reflejar los cambios en el mundo real (Giménez, Casamayor, & Mota, s.f.).

Características principales

Almacenamiento estructurado de datos

Los datos se organizan en tablas, filas y columnas para facilitar su manejo.

Acceso y manipulación eficiente

Las bases de datos permiten consultar y modificar información de forma rápida y ordenada.

Integridad y seguridad de la información

Garantiza que los datos sean precisos, consistentes y estén protegidos contra accesos no autorizados.

Capacidad de realizar consultas complejas

Permite obtener información específica mediante filtros y operaciones avanzadas.

Objetos principales

  • Tablas: Almacenan los datos en filas y columnas.
  • Registros: Filas individuales dentro de una tabla.
  • Campos: Columnas en una tabla que definen atributos de los registros.
  • Índices: Mejoran la velocidad de las consultas.
  • Consultas: Permiten extraer y manipular datos específicos.
